# VScode

### Keyboard shortcuts

|  |  |
| :--- | :--- |
| `option` `W` | wrap selected HTML in `<p>` tags |
| `⌥`  `⇧` `s` | re-order selected CSS properties in alphabetical order |
| `⌥` `⇧` `f` | beautify entire CSS document |
